## 設計原則

### 1. ミニマリズム
- 不要な要素を排除
- 余白を効果的に活用
- 情報の階層化

### 2. 透明性
- 価格の明示
- プロセスの可視化
- 実績の正直な開示

### 3. 知性の表現
- 引用の効果的活用
- 読書への言及
- 思考プロセスの共有

### 4. 機能性
- 直感的なナビゲーション
- 高速な読み込み
- レスポンシブ対応
